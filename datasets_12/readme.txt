this folder contains the regression datasets from PCS_UQ paper (Agarwal et al., 2025)

diamond folder has one-hot encoding for the categorical variables

diamonds_ordinal.csv is the original diamond dataset with no one hot encoding for the categorical variables


References: 
Agarwal, A., Xiao, M., Barter, R., Ronen, O., Fan, B., & Yu, B. (2025). PCS-UQ: Uncertainty Quantification via the Predictability-Computability-Stability Framework. arXiv preprint arXiv:2505.08784.
