This is the repository for the commentary Comparing uniLasso and Lasso: new datasets and new metrics

The analysis folder contains the code for runnning 12 low-dimensionl regression datasets from PCS-UQ paper, 22 datasets from UCI repository (as in uniLasso paper)

For details of each sub-folder, please refer to the readme.MD file in each sub-folder:

datasets_12 folder contains the 12 low-dimensional regression datasets from PCS-UQ paper
datasets_22 folder contains 22 datasets from UCI repository used in uniLasso paper
